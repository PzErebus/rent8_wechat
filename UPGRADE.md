# rent8 微信小程序升级文档

## 项目概述

- **项目名称**: rent8-最常用出租屋管理系统
- **技术栈**: 微信小程序 (TDesign) + NestJS 后端
- **当前版本**: 1.0.0

---

## 一、代码审查建议

### 1.1 后端 (backend)

#### 需要改进的地方：

1. **错误处理** - 建议统一错误响应格式
2. **输入验证** - 建议使用 class-validator 进行参数校验
3. **日志记录** - 建议添加统一的日志中间件
4. **API 文档** - Swagger 文档可以更完善

#### 现有模块：
- `auth` - 认证模块
- `landlord` - 房东管理
- `users` - 用户管理

### 1.2 小程序 (miniprogram)

#### 需要改进的地方：

1. **状态管理** - 建议使用 MobX 或 Redux
2. **代码复用** - 提取公共组件
3. **性能优化** - 图片懒加载、分页加载

#### 现有页面：
- `index` - 首页
- `login` - 登录
- `number` - 房间号管理
- `bill` - 账单管理
- `personal` - 个人中心

## 二、待升级功能

### 2.1 房间管理模块 (number) ✅ 已完成

**功能需求**：
- [x] 房间列表展示（支持分页）
- [x] 添加新房间
- [x] 编辑房间信息（房号、租金、状态）
- [x] 删除房间（软删除）
- [x] 房间状态管理（已出租/空置）

**实现内容**：
- 后端 API: `backend/src/modules/room/`
- 小程序页面: `miniprogram/pages/room/`
  - `room.js` - 房间列表页
  - `add.js` - 添加房间页
  - `detail.js` - 房间详情页

**数据结构**：
```typescript
interface Room {
  id: number;
  roomNumber: string;      // 房号，如 "301"
  floor: number;           // 楼层
  area: number;           // 面积（㎡）
  rent: number;           // 租金（元/月）
  status: 'available' | 'rented';  // 状态
  tenantName?: string;    // 租客姓名
  tenantPhone?: string;   // 租客电话
  createdAt: Date;
  updatedAt: Date;
}
```

**实现内容**：

**数据结构**：
```typescript
interface Bill {
  id: number;

  type: 'rent' | 'water' | 'electric';  // 账单类型
  amount: number;          // 金额
  month: string;           // 月份，如 "2026-03"
  status: 'paid' | 'unpaid';  // 支付状态
  remark?: string;         // 备注
  createdAt: Date;
}
```
